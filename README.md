# Omok (오목)


### < Infomation >
- 2 players. `2인 플레이` 

##### Start by pressing the ReStart button in the top center.

`상단 가운데 ReStart 버튼을 누르면 시작합니다.`

##### 1P is white and 2P is black.

`1P 는 하얀색이고 2P 는 검정색입니다.`

##### First, the person who makes 5 stones wins.

`먼저 5개의 돌을 만드는 사람이 이기는 게임 입니다.`
